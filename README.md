# Around View Parking Assist

ROS와 MORAI 시뮬레이터로 구현한 어라운드 뷰(탑뷰) 주차 보조 프로토타입입니다. 전/후/좌/우 4개 카메라 영상을 호모그래피와 이미지 스티칭으로 변환해 360° 통합 탑뷰 이미지를 생성합니다.

데모 영상(YouTube Shorts): https://youtube.com/shorts/VkxauFfo35o

## 개요

어라운드 뷰 카메라 기반 자율주행 주차 솔루션에서 영감을 받아, 4개 카메라 영상을 탑뷰로 변환·스티칭하는 파이프라인을 엔드투엔드로 구현한 프로젝트입니다.

## 주요 기능

- 탑뷰 변환: OpenCV findHomography, warpPerspective
- 이미지 스티칭 및 중앙 차량 이미지 오버레이
- 측면 경계 크롭으로 왜곡 감소

## 시스템 아키텍처

```
[전방][후방][좌측][우측] 카메라 4개
            |
            v
   [AroundViewNode] --- 영상 수신 / 결과 발행
            |
            v
[AroundViewProcessor] --- 호모그래피 변환 + 스티칭 + 오버레이 + 크롭
            |
            v
   [통합 어라운드 뷰 이미지]
```

## 기술 스택

ROS · MORAI · OpenCV · C++

## 실행 방법

1. MORAI 실행 후 차량에 4개 카메라 피드 연결
2. ROS 노드 실행 → 4개 카메라 토픽 구독
3. rqt_image_view로 출력 시각화

> 호모그래피 정확도는 카메라 배치·화각에 의존하므로 src/dst 포인트 튜닝이 필요합니다.

## 문제점 및 향후 계획

- 연산 부하로 인한 지연, 중첩 영역 이음새 발생
- 향후: 블렌딩 적용, 커버리지 확장, 차선 이탈 경고·주차 슬롯 검출 등 주행 기능 확장
